Preguntas Frecuentes

¿Qué es el mantenimiento 50/50?

Es nuestro método de pago: abonás el 50% al inicio para congelar el precio y comenzar el diseño, y el 50% restante al entregarte la web funcionando.

Mantenimiento "Técnico" (Plan 1)

Asegura que la web no se caiga, protección contra spam y actualizaciones de seguridad del servidor. No incluye cambios de fotos o textos.

$10.000 / mes

Mantenimiento "Activo" (Plan 2)

Incluye todo lo técnico más 1 (una) tanda de cambios de contenido mensual (podes enviarnos nuevas fotos o actualizar un texto de servicios).

$20.000 / mes

Mantenimiento "Evolutivo" (Plan 3)

Pensado para negocios que crecen. Podés pedirnos agregar nuevos botones o secciones pequeñas cada mes para que la web evolucione con tu empresa.

$30.000 / mes

Mantenimiento "Estético" (Plan 4)

Enfocado en la imagen de marca. Incluye rediseños parciales de banners, retoque de colores de temporada y actualización constante de la estética visual.

$40.000 / mes

Mantenimiento "Gold" (Plan 5)

Soporte prioritario 24/7 para el panel de administración, backups diarios de base de datos y optimización de velocidad de carga del servidor dedicada.

$100.000 / mes

¿Qué pasa si quiero dejar de pagar el mantenimiento?

El mantenimiento es necesario para cubrir el costo del servidor y la seguridad. Si se suspende, el sitio quedará fuera de línea. Podés cambiar de plan de mantenimiento en cualquier momento según tus necesidades.
